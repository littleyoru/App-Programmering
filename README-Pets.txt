Project Name:
-------------
Pets


Platform:
---------
React Native, Expo 


Project description:
--------------------
An app that help people find a pet to adopt.

Features:
---------
1. DONE - List information and pictures of pets that can be adopted
2. Search/filter pets
3. Allows people to put up a pet for adoption
4. Use of GPS location to list animal shelters starting from the closest (google maps integration)
5. Account registration and login (optional, with facebook)
